🍕 Pizza Sales Analysis Dashboard

An interactive Power BI dashboard developed to analyze pizza sales performance, customer purchasing patterns, and product trends. The dashboard provides actionable insights to support data-driven business decisions.
1. Short Description / Purpose
The Pizza Sales Analysis Dashboard is a comprehensive Power BI report designed to analyze revenue, orders, customer preferences, and product performance. It helps businesses understand sales trends and optimize menu offerings.
🛠️ Tech Stack
•	📊 Power BI Desktop – Dashboard creation and visualization. 
•	📂 Power Query – Data cleaning and transformation. 
•	🧠 DAX (Data Analysis Expressions) – Measures and calculations. 
•	📝 SQL – Business queries and analysis. 
•	📄 Excel / CSV – Dataset source and preprocessing. 
•	📁 PBIX File – Dashboard development file.
3. Data Source
Pizza sales transactional dataset containing Order ID, Date, Time, Pizza Category, Pizza Name, Size, Quantity, Unit Price, and Total Price.
4. Features & Highlights
Business Problem
Businesses need an efficient way to identify best-selling products, understand customer preferences, and analyze sales trends.
Goal of the Dashboard
To provide interactive insights into revenue, orders, product performance, and customer behavior.
Key KPIs
• Total Revenue: $794,778.85
• Average Order Value: $38.27
• Total Pizzas Sold: 48,176
• Total Orders: 20,769
• Average Pizzas Per Order: 2.32
Business Impact & Insights
• Orders are highest during weekends, especially Friday and Saturday evenings.
• July and January record maximum order volume.
• Classic category contributes the highest sales and orders.
• Large-size pizzas contribute maximum sales.
• Top-selling pizzas drive overall revenue growth.
